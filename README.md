## Compile and install the source

Linux:
```
./gradlew install
```

Windows:
```
gradlew.bat install
```

## IntelliJ IDEA

If you open this in IntelliJ IDEA, in the "Import Project from Gradle" dialog, select "Use customizable gradle wrapper".
